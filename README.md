# Parking Lot Management Application

This project implements a simple Parking Lot Management System using Python.  
The application supports:

- Three types of parking slots: SMALL, LARGE, OVERSIZE  
- Vehicle Parking
- Vehicle removal
- Displaying the status of all parking slots



Features includes:-

- Assign the first suitable open slot to a vehicle based on size compatibility  
- Prevent double-parking of the same vehicle  
- Track which vehicle occupies which slot using an dictionary style lookup  
- Display a clean table of parking lot status  
- Includes seed data for testing  
- Includes automated unit tests
